How to use the runtext api?

Send a GET request to {{ create }} and supply some of the following arguments:

video_text (required)
Text to animate;
duration (optional)
Duration of video in seconds, defaults to {{ default_duration }} seconds;
text_color (optional)
Color of the text, defaults to {{ default_text_color }};
bg_color (optional)
Color of the background, defaults to {{ default_bg_color }}.

For all possible color values and limitations, refer to the ImageMagick color reference.

With this, you can recreate the demo at {{ demo_link }}
with {{ create_demo_link }}.
